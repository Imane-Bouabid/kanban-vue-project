# Kanban Vue.js 3 - Projet Final

## Description

Application **Single Page Application (SPA)** réalisée avec **Vue.js 3**, **Pinia**, **Firebase Authentication** et **Cloud Firestore**.  
Permet de gérer des projets et leurs tâches dans un **Kanban simplifié**.

Fonctionnalités principales :

- Authentification : Email/Password + Google  
- Création / modification / suppression de projets  
- Tableau Kanban avec tâches : À faire / En cours / Terminé  
- Drag & drop pour changer le statut d’une tâche  
- Temps réel avec Firestore  
- Protection des routes pour les utilisateurs non connectés
